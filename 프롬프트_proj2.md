# 정부 안전/노동 관련 주요인사 발언 및 중대재해 분석 웹앱 개발 프롬프트

## 1. 프로젝트 개요
정부 주요 인사의 안전·노동 관련 발언과 국내 중대재해 사례를 수집·분석하고, 포스코그룹의 대응 방향까지 도출하는 Flask 기반 웹 애플리케이션 개발.

---

# 2. 기술 스택

- 백엔드: Python + Flask
- 데이터 수집: Google Gemini API
  - `.env`의 `GOOGLE_API_KEY`
  - 모델: `gemini-2.5-flash`
- AI 분석: OpenAI API
  - `.env`의 `OPENAI_API_KEY`
  - 모델: `gpt-4o-mini`
- 프론트엔드: Bootstrap 5
  - PC/모바일 반응형 지원
- 데이터베이스:
  - SQLite 기본 지원
  - 추후 MySQL/PostgreSQL 전환 가능 구조
- 서버 포트: `5001`
- 운영환경:
  - MacBook
  - Linux 가상서버 호스팅 환경 동시 지원

---

# 3. 환경설정 (.env)

```env
FLASK_APP=app.py
FLASK_ENV=development
SECRET_KEY=change-this-secret

GOOGLE_API_KEY=<키값>
GEMINI_MODEL=gemini-2.5-flash

OPENAI_API_KEY=<키값>
OPENAI_MODEL=gpt-4o-mini

REQUEST_TIMEOUT=12
```

# 4. 핵심 기능 요구사항

## 4-1. 데이터 수집 기능

### 공통 요구사항
- Google Gemini API의 Google Search Grounding 기능 사용
- 데이터 수집은 사용자가 버튼 클릭 시 실행
- 사용자가 기간을 선택할 수 있도록 달력(DatePicker) UI 제공
- 선택된 기간이 프롬프트의 “해당 기간”에 자동 삽입되도록 구현
- 수집 데이터는 DB에 누적 저장
- 담당자가 수정/삭제 가능하도록 구현

---

## 4-2. 주요인사 발언 데이터 수집

### Gemini 프롬프트 1

```text
이재명 대통령과 김영훈 노동부장관이 안전·노동과 관련해 해당 기간에 발언한 내용을 언론 및 소셜미디어(유튜브, X, 페이스북 등) 기준으로 모두 알려주고 출처도 명확히 알려줘.
```

### 데이터 저장 기준
- 기사 1건 = 데이터 1건

### 저장 항목
- 날짜
- 대상인물/기관
- 발언장소
- 주요발언내용
- 핵심키워드
- 기사출처(URL)

---

## 4-3. 중대재해 사례 데이터 수집

### Gemini 프롬프트 2

```text
해당 기간에 발생한 중대재해 사고 사례를 모두 찾아줘.
```

### 추가 요구사항
- 중대재해 기사 발견 시 관련 기사까지 추가 검색
- 연관 기사 기반으로 데이터 확장 수집

### 저장 항목
- 회사명
- 사고일
- 사고내용
- 대외대응
- 시사점
- 기사출처(URL)
- 사과문(담당자 입력 가능)

## 6-1. 종합분석 탭

### OpenAI 분석 기능
- 주요인사발언 데이터 + 중대재해사례 데이터를 통합 분석
- OpenAI API 사용
- 세로형 레이아웃으로 출력

### 분석 항목

#### 1) 주요 시사점
아래 내용을 종합 분석:
- 정부 안전/노동 정책 변화 방향
- 대통령 및 노동부장관 발언 분석
- 타사 산업재해 및 중대재해 사례 분석
- 정부 주요 워딩 및 정책 기조 변화 분석

### 출력 형식 예시

```text
정부의 정책 초점이 ‘대기업 규제’에서 ‘중소규모 사업장 지원 및 예방’ 중심으로 이동

- 정부는 대기업의 안전관리체계는 일정 수준 안착했다고 판단
- 50인 미만 사업장 중심의 예방 및 지원 정책 강화
- AI 기반 중대재해 예방 기술에 대한 관심 증가

☞ 당사 AI 안전기술을 활용한 중소사업장 상생 활동 확대 시 효과 기대
```

### 출력 조건
- `title`, `content`, `direction` 같은 JSON 키 노출 금지
- 보고서 스타일의 자연스러운 텍스트 출력

---

#### 2) 포스코그룹 대응방안
- 주요 시사점 기반 대응 전략 생성
- 경영층 보고용 문체로 출력

---

#### 3) 최근 주요인사 발언 5건

- 기사 URL 정확 연결
- 클릭 시 원문 기사 이동 가능

#### 4) 최근 중대재해 5건
