클로드 계정 2개월 1명
제미나이 API 검색 기능 확인하고 되면 API 유료 결재


[공통사항]
1. 과제명: 그룹 홍보 업무 효율화를 위한 ‘AI 모델 구축’
2. 목표: 대외 부서가 업무에 실질적으로 활용 가능한 그룹 공통 AI 개발
3. 개발 모듈
 - 모듈1: 미디어 인텔리전스(기자 관심사, 기사 기반 소통 가이드 등)
 - 모듈2: 위기대응 지식창고(주요 인사 발언, 중대재해 사례, 사과문 등)
 - 모듈3: 이슈 분석(이슈 핵심 요약, 매체별 논조, 그룹 시사점 등)
4. 개발 요구 조건
 - 초보자 학습/개발 병행으로 유지보수가 쉽게 구현
 - PC 웹 및 모바일 UI 구현-웹앱(Flask + HTML 템플릿) 형태, 복잡하지 않고 실용적인 기능 중심으로 개발
 - RAG와 LLM은 OpenAI API key 사용
 - 환경 관련 내용은 .env 파일에 적용

[모듈1] 미디어 인텔리전스
 1.목적: 홍보 담당자와 기자와의 미팅시 기자별 전문분야, 이력 및 기사 반응 등을 분석하고 미팅시 필요한 대응 능력 제고
 2.분석대상: 네이버 기자홈에 등록되어 있는 기자
 3.주요기능: 기자의 이름과 매체명을 입력하면 네이버 뉴스 API를 통해 데이터를 추출하고 추출된 데이터를 통해서 기자의 관심사항 등을 분석해서 분석 내용과 포스코그룹의 사업회사별 관점의 토킹포인트를 생성
 4.서비스 구현
   - 입력 정보: 기자명, 매체명
   - 출력 정보: 기자명, 매체명, 직책, 출입분야, 학력, 출신지역, 경력, 출생년도, 휴대폰번호, 이메일, 최신기사(30건), 관심분야, 기사논조, 최근 관심 키워드, 관련기사, 기자와 미팅시 토킹포인트
     * 관련 기사: 포스코그룹 관련 기사(최신순, 30건), 댓글 반응이 좋은 기사(최신순, 5건)
                -> 관련 기사 중에서 '단독' or '기획' or '르포' or '기자수첩' or '논설' or '사설' or '오피니언' 등은 별도 표시
   - 토킹포인트: 포스코그룹 사업회사별 기자 관심사를 반영한 대화 주제 및 이슈에 대해 내용으로 생성 

[모듈2] 정부 정책 동향 분석
 1.목적: 정부 정책 동향과 부합하는 홍보 방향 수립 
 2.분석대상: 네이버 뉴스 -> API에서 전체문서 나오면 사용하고 안나오면 제미나이에서 검색
 3.주요기능: 분석대상에서 주요 정부 인사가 발언한 내용을 추출하고 분석
   - 네이버 뉴스: 키워드(중대재해, 안전사고, 노란봉투법, 원하청)로 네이버 API 에서 데이터 추출
      * 검색 기간은 설정 가능하도록 구현
   - RAG 시스템
     * 국정 5개년 계획(pdf 문서) -> 제미나이 API에서 검색되면 검색하는거로 하고
   - 출력 정보: 안전, 환경, 노동, 산업 등 정부 정책동향을 국정 5개년 계획과 연계하여 분석한 정보, 분석에 따른 포스코그룹 시사점 및 대응방안
     * 정부 정책동향: 날짜, 대상 인물/기관, 발언 장소, 주요 발언 내용, 핵심 키워드, 주요 발언 기사 출처
     * 포스코그룹 시사점 및 대응방안: 정부 정책동향 분석을 통한 포스코그룹 차원에서의 시사점 및 대응방안 생성

[모듈3] 이슈 분석
 1.목적: 관심 키워드에 대한 최신기사 기반 사업회사 관점의 이슈분석 보고서 제공
 2.분석대상: 네이버 뉴스, 네이버 데이터랩, 과거 회사 자료(입장문, 보도자료 등, RAG 시스템) 
 3.주요기능: 관련 키워드를 입력하면 분석대상에서 데이티를 추출하고, 데이터 분석을 통해 사업회사별 이슈분석 및 인사이트 리포트 생성
   - RAG 벡터 DB에 회사 관련자료를 저장
   - 최신 데이터는 네이버 뉴스, 네이버 데이터랩에서 검색
   - 키워드로 검색시 최신 데이터와 RAG 시스템 데이터를 활용하여 회사의 입장 메세지, 숨겨진 히든 리스크, 부족한 개선점 발굴 내용을 생성
